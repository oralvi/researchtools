# DCPD GUI - Rust + Tauri

Rust + Tauri 实现的 DCPD 图形界面应用。提供专业的桌面应用体验。

## 快速开始

### 开发模式运行
```bash
cd DCPDexport/src/gui/rust
cargo tauri dev
```

### 构建发布版本
```bash
cargo tauri build
```

## 功能

- 📊 数据分析（支持 CSV/DAT/XLSX）
- ⚙️ 文件处理（单个/批量）
- 🔍 系统监控和日志
